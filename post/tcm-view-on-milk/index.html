牛奶喝多病痛越多，正確喝牛奶方法！

牛奶寒性 痰濕 雌激素 骨質疏鬆 脾陽 乳糖不耐 牛奶正確喝法 子宮肌瘤

牛奶在西方健康觀念中的迷思，從中醫角度分析屬性及對人體的影響，強調了過量飲用牛奶可能導致的問題，並提供了如何正確飲用牛奶的建議。

以下為精簡筆記#

  • 西方觀念：

    • 牛奶 = 健康必需品 → 小孩長高、老人防骨鬆。

    • 實際：中醫認為牛奶性微寒，非人人適合。

  • 中醫觀點：

    • 入心肺胃經。

    • 功效：補虛贏、潤皮膚、養胃陰、強筋骨。

    • 關鍵：微寒 → 適量可益，長期必害。

  • 現代牛奶問題：

    • 人工激素催乳 → 含大量雌激素。

    • 後果：婦女雌激素失衡 → 子宮肌瘤、乳癌、卵巢囊腫。

    • 西醫僅手術、化療，不談根源。

  • 取象比類：

    • 乳牛：大腹細腿、少動。

    • 人：長期飲用 → 大肚、四肢無力、懶動。

  • 痰濕問題：

    • 牛奶寒潤黏膩 → 傷脾陽。

    • 脾主運化 = 鍋爐需火 → 牛奶如冷水澆滅火。

    • 結果：消化失常 → 營養成垃圾 → 腹部堆積脂肪。

    • 痰濕上壅肺 → 喉中痰卡、白痰稀濕、昏沉沉重。

  • 補鈣迷思：

    • 牛奶確含鈣，但吸收需腎陽推動。

    • 長期飲用 → 傷脾腎陽 → 無法吸收 → 多餘鈣排出。

    • 結果：骨鬆未解 → 反得腎結石。

  • 不同體質反應：

    • 多數人：肥胖、痰濕重。

    • 少數人：消瘦不胖 → 脾胃虛弱，無法吸收，牛奶直瀉。

  • 適合人群：

    • 胃陰虛、脾不弱者。

    • 表現：舌紅少苔/剝落苔、口燥便乾、胃灼熱、飲水不解渴。

  • 現代佐證：

    • 亞洲人多缺乳糖酶 → 乳糖不耐受。

    • 脹氣、腹痛、腹瀉。

    • = 中醫所言「脾虛不能運化水飲」。

  • 正確飲用方法：

    • 不可空腹。

    • 配合五穀雜糧（健脾溫平）。

    • 正餐後 1-2 小時飲用。

    • 最佳：加入健脾祛濕粉（山藥、薏米、茯苓、芡實、白扁豆）。

      • 中和寒性 → 助消化吸收 → 減少乳糖不耐受。

  • 總結：

    • 牛奶非神藥。

    • 養生關鍵：依體質、懂藥性、巧搭配、知制衡。

    • 大道：吃得對、吃得巧 → 體質與環境和諧。

以下內容為倪海廈語音檔本文#

我跟諸位來談一個東西，牛奶，這個牛奶啊， 現在被西方的這個健康觀念洗腦洗得非常厲害， 好像每天早上不喝一杯牛奶， 你這個人就不健康， 小孩子不喝牛奶就長不高， 老人不喝牛奶就骨質疏松， 講的跟神仙妙藥一樣， 我告訴你，這完全是胡說八道， 這里面問題非常大， 這個東西或許它適合西方人的精神， 這個我不去評論， 但是拿來硬套在各位的身上， 那問題就出來了， 為什麽？ 各位要先了解中醫是怎麽看這個牛奶的， 中醫的要點里面， 你去看，牛奶，胃肝，性微寒， 這個微寒兩個字就是關鍵所在， 它歸心肺胃經，有什麽功效呢？

補虛贏，潤皮膚，養胃陰，強筋壯骨， 你看功效寫的這麽好， 補虛贏，就是你這個人很瘦弱， 氣血很差，喝一點能夠補足，補起來。 潤皮膚，皮膚很乾燥， 喝的能滋潤，聽起來都很好，對不對？ 所以大家一看， 哇，這麽棒，就開始天天喝， 長期喝，把這個當水喝， 我告訴你，這絕對不是養生之道，

過猶不及啊！任何東西都要有一個度， 你偶爾喝，適量喝， 在你身體需要它的時候喝，沒有問題， 可是你把它當成一個每天必須的營養品， 長期不斷的灌到身體里面， 那你就準備出問題了。

諸位要想一個問題，現在市面上的牛奶， 你以為還是以前那種，一個母牛，生的小牛， 它多出來的奶水給人喝嗎？ 早就不是了，現在的牛奶是怎麽來的， 是打雌激素刺激出來的， 那個母牛根本就沒有懷孕， 或者是一直處在一個懷孕跟產乳的循環里面， 用大量的荷爾蒙去刺激它秘乳， 你想想看，這種充滿了人工激素的東西，它能好嗎？

你把這些帶有大量雌激素的東西喝到肚子里面去， 尤其是婦女同胞， 你體內的雌激素本來就有一個平衡， 現在你每天從外面灌進來一大堆， 這個雌激素在體內過度堆積， 直接就往哪里跑， 就往你的子宮，你的乳房跑， 所以現在為什麽那麽多的子宮肌瘤，乳癌，卵巢囊腫， 你以為這怎麽來的？ 很多就是這樣吃出來的，喝出來的， 這個東西西醫它不告訴你，它只會叫你定期檢查， 長了東西就開刀，割掉，化療， 它從來不告訴你源頭在哪里， 源頭就在你的飲食里面。

然後中醫還講究一個東西，叫做取象比類， 這個非常重要，是老祖宗的智慧， 你看那個東西是什麽樣子，你吃多了， 你人就容易變成那個樣子， 你去看現在的乳牛是什麽樣子，肚子好大一個， 脹的跟氣球一樣，然後四個腿細細的，對不對， 而且現在的乳牛都是圈養， 關在那個小小的格子里面， 動的不能動， 它一輩子可能都沒走過幾步路， 你想一下，你長期喝這個牛奶， 你是不是也變成那個樣子，肚子越來越大， 四肢越來越沒有力氣，而且人越來越懶， 動都不想動，吃飽了就想躺著， 就想窩在沙發上， 你看，那不是跟那個牛一個樣子的嗎？

很多長期喝牛奶的人， 尤其是小姐們， 天天喊著要減肥， 跑來問我， 老師啊！我這個肚子怎麽都瘦不下來？ 我三餐吃很少啊？怎麽還是胖？ 我問他，你是不是每天早上都喝一杯牛奶或者拿鐵？ 他說是啊，不是很健康嗎，我說這能怎麽減，根本減不下來嘛， 你那個造成肥胖的源頭都沒有切斷， 你後面做再多努力都是白費力氣， 好不好，為什麽會這樣子。

因為牛奶這個東西，它會助長，痰濕， 這兩個字，中文一定要記在心里面， 什麽是痰濕？就是身體里面代謝不掉的垃圾， 牛奶性危寒，又那麽滋潤，黏膩， 你喝下去，第一個傷害的就是你的脾陽，

中醫講脾主運化，你的消化系統就像一個鍋爐， 需要火去燒，這個火就是脾陽， 現在你一杯冷冰冰又黏膩的牛奶灌下去， 等於是一瓢冷水直接澆在那個火上面， 那個火是不是就弱掉了， 脾陽被這個痰濕一困住， 你的消化系統怎麽能好了起來？ 它沒有能力去運化這些東西的， 那吃進來的營養怎麽辦？ 就變成半成品，變成垃圾，堆在身體里面， 堆在哪里，最喜歡堆在肚子上， 所以你的脂肪就開始堆積， 一層又一層，肚子就越來越大， 然後這個痰濕還會往上跑，

中醫講脾為生痰之源，肺為貯痰之器， 意思就是說這個痰溼脾胃消化不良產生的， 但是儲存的地方是在肺里面， 所以你會發現很多長期喝牛奶的人， 喉嚨老是覺得不舒服，好像有東西卡在那里， 咳不出來，也吞不下去，動不動就要咳，清一下喉嚨， 而且咳出來的痰多半是那種濕痰，白痰，很稀的那種， 這就是標準的寒濕之痰。 你身體里面積了一堆濕氣跟垃圾， 你人怎麽會清爽？ 每天都昏昏沈沈的，頭重腳重， 像裹著一條濕棉被一樣， 這都是痰濕困脾的標準症狀。

好，接下來講到一個最大的迷思，就是補鈣， 大家都說喝牛奶能補鈣，是能補鈣， 這個我不否認，畢竟牛奶雖然是催化出來的， 但它也是牛的精血所化身的東西， 它確實有強筋骨補肝腎的作用， 所以說它能補鈣，這個理論上沒錯， 但是你有沒有想過，它補的鈣，到底補到哪里去了？

人體有那麽聰明，你喝下去的鈣，它就那麽精準的， 啪一下，全部補到骨頭里面去嗎？ 如果真的能，那怎麽解釋現在全世界喝牛奶最多的國家， 像是美國，北歐那些國家， 他們的骨質疏鬆率也是全世界最高的，這不是很矛盾嗎？ 他們從小說牛奶是主食啊， 照理說，骨頭應該跟石頭一樣硬才對啊！ 怎麽反而最容易疏鬆？ 最容易骨折，這就說明一個問題， 你吃進去的鈣跟你身體能不能吸收， 能不能利用，是兩回事， 好不好，你身體要能利用這個鈣， 需要陽氣去推動，需要腎臟去主導， 腎主骨頭的強壯是腎臟在管的， 結果你長期喝這種性寒的牛奶， 不但傷了脾陽，連帶也傷了腎陽， 你腎的陽氣都不足了，功能都減退， 他哪有能力去把這些鈣抓過來， 好好的放到骨頭里面去，根本沒有辦法， 那這些多餘的身體用不掉的鈣， 最後都得靠腎臟把它排出去。

你想想看，一個骨質疏鬆的人， 本身他的腎功能就已經在退化了， 你還給他這麽重的負擔， 讓他去處理這些他根本用不掉的鈣， 最後的結局是什麽？ 骨質疏鬆一點都沒解決，反而得了腎結石。

這個鈣亂跑，跑到腎臟里面就變成石頭了嘛， 你去照X光，腎臟里面一顆一顆亮亮的都是鈣結石， 吃鈣片也是一模一樣的道理， 你吃進去一大堆，身體不能用， 最後就是給自己找麻煩，對不對？

當然，長期喝牛奶的人， 還有另外一種體型，就是特別瘦， 怎麽吃都吃不胖，這種人他跑來跟我說， 老師啊，你看我喝牛奶都沒胖， 是不是代表我沒事? 我說你這個問題可能更大， 為什麽？ 因為你的脾胃， 已經虛弱到連消化牛奶這種高能量的東西的能力都沒有， 你喝下去，它根本沒有被吸收， 就直接穿腸而過，變成腹瀉排出去了。

營養一點都沒進到身體里面，你當然不會胖， 這種人雖然沒有變胖， 但是他的脾胃功能是處在一個非常虛弱的狀態， 這種其實也不好，只是沒有堆積成痰濕而已。

那麽，到底什麽樣的人可以喝牛奶呢？#

古書上寫的很清楚， 唐代藥王孫思邈在他的《千金翼方》里面記載， 牛乳為寒補虛贏，止渴。 所以中醫認為， 牛奶最適合的是那種胃陰虛而脾不弱的人飲用， 這句話是關鍵。 什麽叫胃陰虛？ 就是你胃里面那個汁潤的津液不夠了， 好像鍋子里面的水快燒乾。

這個時候牛奶色白，質地粘稠， 喝下去剛好能夠養胃陰，補胃液， 但是前提是你的脾氣要健運， 你的脾臟功能是好的，能夠運化這個水飲。 這個時候你喝牛奶就相當棒了。 那怎麽知道自己是不是胃陰虛呢？ 很簡單，你把舌頭伸出來看一下， 如果你的舌頭很紅，上面幾乎沒有舌苔， 或者舌苔一塊一塊地剝落， 像地圖一樣，這就叫地圖，舌剝落苔。 症狀上呢，你會覺得胃里面隱隱的有灼熱感。 老是覺得口甘舌燥，很想喝水， 但是喝的水也不解渴， 大便還很乾結。 如果你有這些症狀，舌頭又是那個樣子。 好了，各位可以記筆記了。 這個時候你喝點牛奶，就是對症下藥。

現代醫學的研究，其實也從側面證實了這一點。 研究發現，亞洲人的體內，普遍都缺乏足夠的乳糖酶。 這個乳糖酶是幹嘛的？ 就是用來分解牛奶里面的乳糖的。 你沒有這個酶，乳糖就沒辦法被分解， 它就會滯留在你的腸道里面。 然後，腸道里面的細菌一發酵，就產生大量的氣體。 所以你就會脹氣、腹痛、咕嚕咕嚕叫， 最後就腹瀉。這就是所謂的乳糖不耐受。

你看， 這不就跟中醫講的脾虛不能運化水飲的道理， 不謀而合嗎？ 只是講法不一樣而已， 但指的都是同一件事情。

所以，如果你真的又嘴饞， 想透過牛奶攝取一些優質蛋白， 又不想損傷你的脾胃， 不想喝完之後出現腹瀉、腹漲，口氣很大， 整天疲勞困重，嘴巴黏黏膩膩， 人不清爽，這些痰濕的症狀。

那怎麽辦呢？ 我教各位幾個方法， 絕對不要空腹喝牛奶， 空腹的時候，你的胃里面是空的， 牛奶一下去，直接刺激腸胃， 乳糖很快就抵達腸道。 乳糖不耐受的癥狀馬上就出來了， 你最好在喝牛奶的同時， 配上一些麵包、餅乾、饅頭之類的澱粉類主食。 為什麽？ 從中醫角度來講，五谷雜糧這些東西， 都是入脾經的，性味多是溫平的，都能夠健脾。 你用這些健脾的東西，先給脾胃打個底，墊一下， 它就有能力去處理後面進來的牛奶了。

你可以在正餐之後， 隔個一兩個小時再喝， 這個時候胃里面還有食物， 可以延緩牛奶排空的速度， 讓它有更充足的時間被消化。 但是，這都還只是治標的方法， 最好的方法是什麽？ 是直接在牛奶里面加入能夠健脾， 又能夠對抗它寒性的東西。

這就是中醫君臣佐使配伍的智慧了。 你把牛奶當成君藥，雖然它有偏性， 但是你可以用臣藥、佐藥去制衡它，幫助它。

比如說，像我常常提到的健脾八珍粉， 這個東西非常好用， 里面的成分全部都是藥食同源的，非常平和。

像山藥、薏米、茯苓、芡實、白扁豆這些。 你在家里可以自己去買這些材料， 用調理機把它們打成很細的粉末。 喝牛奶的時候，就挖一兩勺這個粉， 用熱牛奶把它沖開，攪拌均勻的再喝。 你看這樣多好。 山藥補脾益氣， 薏米健脾利濕，茯苓也是利水滲濕的， 芡實能固腎澀精，白扁豆能化濕和中。 這些東西加在一起， 本身就是一個非常好的健脾祛濕方。

它們的性質是偏溫的， 能夠直接對抗牛奶的那個微寒之性， 把它中和掉。 同時，這些藥食兩用的食材還含有非常豐富的膳食纖維， 比你吃那些精米白面做的主食， 更能有效的延長牛奶在腸胃里停留的時間。 這樣一來，不僅大大減緩了乳糖不耐受的發生， 還幫助了牛奶里面那些優質蛋白的吸收和利用。

這不就是為各位的身體高效獲取營養提供了最有力的保障嗎? 所以，各位啊， 養生不是盲從， 不是聽別人說什麽好， 你就跟著吃什麽。 你要先了解你自己的體質， 了解食物的藥性， 我們老祖宗的智慧就在這里。 他不是叫你什麽都不能吃， 而是叫你怎麽吃，吃得對，吃得巧， 學會如何去搭配，去制衡。 這才是真正的養生大道。
